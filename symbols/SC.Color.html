Class: SC.Color


Extends SC.Copyable, SC.Object.

Provides colorspace conversions between rgb and hsl.

This object can be instantiated by using create if it's a simple RGB color, or through SC.Color.from, which will turn any valid CSS color into it's appropriate SC.Color.

To get the CSS value of the color, call toCSS, which will provide the best CSS color to use according to browser support. This means that in IE, colors with an alpha channel will fall back to use ARGB, which requires the following hack to use:

// This hack disables ClearType on IE!
var color = SC.Color.from('rgba(0, 0, 0, .5)').get('cssText'),
    css;
if (SC.Color.supportsARGB) {
  var gradient = "progid:DXImageTransform.Microsoft.gradient";
  css = ("-ms-filter:" + gradient + "(startColorstr=%@1,endColorstr=%@1);" +
         "filter:" + gradient + "(startColorstr=%@1,endColorstr=%@1)" + 
         "zoom: 1").fmt(color);
} else {
  css = "background-color:" + color;
}

You may want to use the sub, add, and mult functions to tween colors between a start and end color.

For instance, if we wanted to tween between the color "blue" and "teal", we would to the following:

var blue = SC.Color.from("blue"), teal = SC.Color.from("teal"), delta = blue.sub(teal);

// Tick is called using a percent // between 0 and 1 function tick (t) { return blue.add(delta.mult(t)).get('cssText'); }
