Class: SC.SheetPane


Extends SC.PanelPane.

Displays a modal sheet pane that animates from the top of the viewport.

The default way to use the sheet pane is to simply add it to your page like this:

SC.SheetPane.create({
  layout: { width: 400, height: 200, centerX: 0 },
  contentView: SC.View.extend({
  })
}).append();

This will cause your sheet panel to display. The default layout for a Sheet is to cover the entire document window with a semi-opaque background, and to resize with the window.
